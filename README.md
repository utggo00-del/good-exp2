# 참체험 🌊 — 시흥시 어린이 체험 프로그램 가이드

시흥시 아이와 함께 갈 수 있는 **체험 프로그램 107개+** 를 한눈에 찾아보는 싱글페이지 앱(SPA)입니다.

## 주요 기능

- **나이 필터** — 아이 나이를 선택하면 맞는 프로그램만 표시
- **체험 분야 필터** — 자연·생태 / 해양·과학 / 역사·문화 / 교육·안전 / 스포츠·레저 / 공예·예술
- **빠른 필터** — 무료만 / 상시예약 / 방학한정 / 물놀이 / 키즈카페 / 서핑·수상레저 등
- **실시간 운영상태** — Supabase 연동으로 현재 모집중 / 마감 / 준비중 상태 표시
- **체험 완료 도장판** — 다녀온 프로그램에 도장을 찍어 진행상황 관리
- **예약 사이트 바로가기** — 각 프로그램별 공식 예약 링크 직접 연결

## 시흥 특색 프로그램

| 카테고리 | 대표 프로그램 |
|---|---|
| 갯골·염전 | 갯골생태공원 염전체험, 수상자전거, 전기차투어 |
| 서핑·수상레저 | 웨이브파크 서핑레슨, 해양레저아카데미 (무료) |
| 물놀이 | 배곧한울공원 해수체험장, 시흥시 18개 공원 물놀이장 |
| 갯벌 | 오이도 어촌체험휴양마을 갯벌체험 |
| 키즈카페 | 너티차일드, 공룡월드, 브레드이발소타운 등 11개 |

## 기술 스택

- **Frontend** : Vanilla HTML / CSS / JavaScript (프레임워크 없음)
- **Backend** : [Supabase](https://supabase.com) — `program_status` 테이블로 운영상태 관리
- **Auth** : 닉네임 + PIN 방식 커스텀 로그인 (관리자 상태 업데이트용)
- **Hosting** : GitHub Pages
